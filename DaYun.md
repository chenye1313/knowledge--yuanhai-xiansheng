# 调用条件：用户选B或问大运、流年、今年、明年

# 方法论（先看这里）
# ═══════════════════════════════════════════════════════════════

methodology:
  
  分析步骤:
    第一步: "确定当前大运"
    第二步: "分析大运吉凶"
    第三步: "分析流年吉凶"
    第四步: "给出建议"

  排大运顺逆_硬规则:
    使用方式: "只能按四格表查，不要用临场推理替代表格"
    阳年: "年干为甲、丙、戊、庚、壬"
    阴年: "年干为乙、丁、己、辛、癸"
    四格表:
      男命阳年: "顺行"
      男命阴年: "逆行"
      女命阳年: "逆行"
      女命阴年: "顺行"
    简记: "男阳女阴顺行，男阴女阳逆行"
    绝对禁止:
      - "禁止把女命阴年判为逆行"
      - "禁止看到女命就默认逆行"
      - "禁止看到阴年就默认逆行"
      - "禁止在表格写顺行，却在结论写逆行"
    回归测试:
      1985乙丑女命: "乙为阴干，女命阴年，必须顺行；判逆行就是错"
      1990庚午男命: "庚为阳干，男命阳年，必须顺行"
      1990庚午女命: "庚为阳干，女命阳年，必须逆行"
      1985乙丑男命: "乙为阴干，男命阴年，必须逆行"

  起运节气_硬规则:
    顺行: "从出生日时数到下一个「节」"
    逆行: "从出生日时数到上一个「节」"
    注意: "这里用的是「节」，不是中气；小满、夏至、大暑、雨水等中气不能用于起运"
    搜索模板: "搜索「{出生年}年 {目标节} 具体时间」；目标节必须由顺逆和当前节气月份决定，不能固定写芒种"

  起运年龄_换算规则:
    时间差: "从出生时刻到目标「节」的时间差，尽量精确到小时"
    换算: "3天=1岁，1天=4个月；不足1天按小时折算"
    输出必须包含:
      - "顺行/逆行"
      - "目标节名称与公历时间"
      - "出生时刻到目标节的时间差"
      - "起运年龄"
      - "交运公历年月"
    禁止:
      - "只算整天、不看小时"
      - "查错节气后继续计算"
      - "把顺行规则套到逆行命局"

  大运干支_排列规则:
    顺行: "第一步大运从月柱的下一柱开始，之后每十年顺推一柱"
    逆行: "第一步大运从月柱的上一柱开始，之后每十年逆推一柱"
    禁止:
      - "禁止把月柱本身当成第一步大运"
      - "禁止只写10-19岁、20-29岁这类整十档，掩盖实际起运月"
    年龄段: "以实际起运年龄或交运年月为起点，每10年一柱；例如10岁4个月起运，就写10岁4个月-20岁4个月"
  
  第一步_确定大运:
    方法: "根据排盘结果，找到当前年龄对应的大运"
  
  第二步_大运分析:
    看天干: "大运天干与命局关系"
    看地支: "大运地支与命局关系"
    看喜忌: "是喜用神还是忌神"
  
  第三步_流年分析:
    看流年干支: "流年与命局、大运的关系"
    看冲合: "是否冲动命局"
    看喜忌: "流年是喜是忌"
  
  输出模板: |
    【顺逆查表】年干阴阳 + 性别 → 顺行/逆行
    【起运计算】目标节、时间差、换算过程、起运年龄
    【当前大运】X岁-X岁 某某运
    【大运评价】吉/平/凶，原因
    【今年流年】某某年
    【流年评价】吉/平/凶，原因
    【明年预测】简要分析
    【建议】具体可操作建议
