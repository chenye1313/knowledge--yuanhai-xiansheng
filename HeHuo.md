# HeHuo - 商业合伙知识库
# 调用条件：用户问合伙、入股、创业合作、共同做生意、股权、分红、退出机制


# ═══════════════════════════════════════════════════════════════
# 方法论（先看这里）
# ═══════════════════════════════════════════════════════════════

methodology:

  核心原则: |
    商业合伙不是婚恋合盘，不能只用生肖、日干相合来判断。
    必须同时看：双方命盘、财星/比劫、事业分工、大运流年、现实合伙机制。
    信息不完整时只能做粗判，不能给正式评分。

  必要信息:
    自己: "出生年月日时、性别、出生地、已排命盘"
    对方: "出生年月日时、性别、出生地；若只有农历，必须先联网换算公历并查日柱"
    项目: "行业方向、投入金额、股权比例、各自负责事项、是否影响主业"

  分析步骤:
    第一步: "先排双方命盘；缺少对方时辰时，只能做三柱粗判"
    第二步: "判断各自财星、比劫、官杀、印星状态"
    第三步: "判断双方资源是否互补，还是同质竞争"
    第四步: "结合当前大运和流年，判断是否适合落地"
    第五步: "给出合作方式、风控条款和是否评分"

  必调知识:
    财务风险: "必须调用 CaiYun.md"
    事业分工: "必须调用 ShiYe.md"
    时机判断: "必须调用 DaYun.md"
    喜忌补救: "必要时调用 XiJi.md"
    婚恋合盘: "只有用户问感情、婚姻、伴侣时才调用 HeHun.md"

  联网白名单:
    允许联网:
      - "农历转公历"
      - "日柱万年历"
      - "节气时间"
      - "流年干支"
      - "今日干支"
    禁止联网:
      - "搜索某日主财运分析后当成结论"
      - "搜索合伙创业八字分析后当成知识库"
      - "用外部命理文章替代本知识库判断"


# ═══════════════════════════════════════════════════════════════
# 评分规则
# ═══════════════════════════════════════════════════════════════

scoring:

  正式评分前置条件:
    - "双方完整四柱已经排出"
    - "双方日柱已经联网万年历确认"
    - "项目行业、出资、分工、退出机制至少有基本信息"
    - "已经调用商业合伙、财运、事业、大运流年相关知识"

  信息不足处理:
    无对方时辰: "只给粗判区间，不给正式分数"
    无对方日柱: "停止精准合伙判断，先补日柱"
    无行业和股权: "只判断命理配合，不判断商业可行性"

  评分维度:
    命盘互补度:
      权重: 25
      看点: "日干关系、五行互补、用神是否互补"
    财务风险:
      权重: 25
      看点: "财星强弱、比劫夺财、财星受冲克、资金回收难度"
    事业分工:
      权重: 20
      看点: "官杀、食伤、印星、执行/管理/资源分工是否清楚"
    运势时机:
      权重: 20
      看点: "当前大运流年是否支持落地、是否冲合关键柱"
    现实机制:
      权重: 10
      看点: "协议、账目、退出、试合作、主业影响"

  分数解释:
    80_100: "可以合作，但仍需书面协议和试运行"
    65_79: "可以小额试合作，不宜重仓绑定"
    50_64: "谨慎，只适合项目制、短周期合作"
    0_49: "不建议合伙，最多保持资源互换"


# ═══════════════════════════════════════════════════════════════
# 判断规则
# ═══════════════════════════════════════════════════════════════

knowledge:

  比劫与合伙:
    比劫旺:
      含义: "合伙缘、人脉、同辈资源强"
      风险: "利益分配、抢单、抢资源、账目不清"
      建议: "必须白纸黑字，设置退出机制和财务边界"
    比劫弱:
      含义: "合伙推动力弱，适合借力但不宜完全依赖别人"
      建议: "合作要以资源交换为主，避免承担过多义务"

  财星与出资:
    财星旺且身能任财:
      判断: "更适合承担资金、商务、成交角色"
      风险: "仍需看比劫是否夺财"
    财星弱:
      判断: "不宜大额出资或替别人垫钱"
      建议: "先用能力、渠道、试点项目换收益，不急着入股"
    财星入库或受冲:
      判断: "钱容易被套住或回款慢"
      建议: "合同里必须写清回款、分红、退出和坏账承担"

  官杀与主业:
    官星年:
      含义: "主业、职位、责任、规则被放大"
      建议: "主业上升期不要轻易分散核心精力"
    七杀重:
      含义: "压力、竞争、突发情况多"
      建议: "可以谈判和试合作，不宜仓促重仓"

  印星与准备:
    印星有力:
      含义: "学习、资质、贵人、系统建设更有利"
      建议: "适合补课、建规则、搭团队、完善协议"
    印星弱:
      含义: "支撑系统不足"
      建议: "先补流程、财务、合同和复盘机制"

  同龄同生肖:
    优势: "成长背景相近，沟通成本低"
    风险: "资源圈层可能重叠，决策容易互不相让"
    处理: "看月柱、日柱、行业资源和现实分工，不可只凭生肖下结论"
