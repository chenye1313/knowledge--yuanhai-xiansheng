# JianKang - 健康分析知识库
# 调用条件：用户选F或问健康、身体、疾病


# ═══════════════════════════════════════════════════════════════
# 方法论（先看这里）
# ═══════════════════════════════════════════════════════════════

methodology:
  
  分析步骤:
    第一步: "分析五行偏枯"
    第二步: "查天干地支对应易感部位"
    第三步: "找需要留意的年份"
    第四步: "给出生活方式建议"
  
  第一步_五行偏枯:
    方法: "统计命盘中五行分布"
    太过: "某五行太多，对应部位或习惯倾向需要留意"
    不及: "某五行太少，对应部位或恢复力倾向需要留意"
    被克: "某五行被克严重，对应主题需要多观察，不作疾病诊断"
  
  第二步_对应部位:
    查表: "用下方天干地支易感部位表"
    原理: "哪个干支受冲克或失衡，对应部位作为生活提醒，不直接断病"
  
  第三步_注意年份:
    方法: "大运流年冲克命盘忌神或用神时"
    信号:
      - "用神被冲克"
      - "忌神太旺"
      - "岁运并临"
  
  输出模板: |
    【健康概况】一句话总结
    【五行分析】哪个五行偏枯，对应什么脏器
    【易感部位】根据干支表推断
    【注意年份】哪些年份要注意健康
    【养生建议】具体可操作建议
    
    ⚠️ 命理分析仅供参考，健康问题请咨询医生。

  健康建议边界:
    非医疗诊断: "命理健康分析只能说易感部位、生活习惯倾向和注意年份，不能预测疾病、诊断疾病或给治疗方案。"
    不等于开运补救: "健康养生建议是生活方式提醒，不可直接等同于喜忌补救。"
    颜色食物要过喜忌: "如果输出颜色、方位、饰品、开运食物，必须先用喜忌规则校验，不能按五行偏枯直接推荐。"
    医疗优先: "涉及症状、疾病、用药、治疗时，必须建议咨询医生，不给诊断或治疗方案。"
