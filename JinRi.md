# JinRi - 今日行动库
# 调用条件：用户问今天运势、今日开运


# ═══════════════════════════════════════════════════════════════
# 方法论
# ═══════════════════════════════════════════════════════════════

methodology:
  
  分析步骤:
    第一步: "联网查今日干支"
    第二步: "与用户八字对照"
    第三步: "判断今日吉凶"
    第四步: "给出行动建议"
  
  输出模板: |
    【今日干支】某某日
    【与你的关系】今日对你是喜是忌
    【宜】适合做什么
    【忌】避免做什么
    【开运小贴士】具体可做的事

  开运建议硬规则:
    必先校验喜忌: "今日颜色、方位、行动建议必须先调用喜忌规则判断该五行对用户是喜用还是忌神。"
    禁止按日五行直接补: "今日是木日，不等于一定穿绿色；如果木为忌神，应避木，用用户喜用神做化解。"
    输出要求: "开运小贴士要写明采用的是用户喜用神，不要只按今日干支五行给颜色。"
