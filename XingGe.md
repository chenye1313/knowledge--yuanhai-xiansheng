# 方法论（先看这里）
# ═══════════════════════════════════════════════════════════════

methodology:
  
  分析步骤:
    第一步: "看日干（核心性格）"
    第二步: "看日支（内心世界）"
    第三步: "看十神配置（性格倾向）"
    第四步: "看五行强弱（性格偏向）"
    第五步: "综合给出性格画像"
  
  第一步_日干性格:
    方法: "根据日干查十干体象"
    原理: "日干是自己，日干的五行属性决定核心性格"
  
  第二步_日支内心:
    方法: "根据日支查日支含义"
    原理: "日支是内心，代表深层性格和配偶特征"
  
  第三步_十神配置:
    方法: "看命盘中哪个十神最旺"
    原理: "旺的十神会强化某种性格特质"
  
  第四步_五行强弱:
    方法: "看五行偏枯"
    原理: "五行对应性情，哪个强哪个特质明显"
  
  输出模板: |
    【核心性格】日干决定的基本性格
    【内心世界】日支代表的内在特质
    【性格特点】十神带来的性格倾向
    【优点】2-3个优点
    【需注意】1-2个需要注意的地方
    【相处建议】如何与这种性格的人相处
